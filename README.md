# photo-stash

Side project made with Next JS 13. Used only frontend technologies and zero backend. For storing liked photos, browser's window.localStorage was used. 
Live link of working [app](https://photo-stash.vercel.app/) is deployed and hosted at [Vercel](https://vercel.com/).

This project reaches performance A grade in [GTMetrix](https://gtmetrix.com/) performance tracking website.

![Screenshot of GTMetrix performance scores for Photo Stash](/public/images/actions/screenshot_of_gtmetrix_performance_grade.jpg)

You can clone this repo and run standart npm commands to start developing at local environment:
```
npm install

npm run dev
```
